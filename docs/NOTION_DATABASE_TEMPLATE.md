# Notion 数据库模板

## 创建步骤
1. 在 Notion 创建新数据库
2. 添加以下列：

| 列名 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 金额 | Number | ✅ | 记账金额 |
| 类型 | Select | ✅ | 支出 / 收入 |
| 分类 | Text | ✅ | 支出分类，自由填写 |
| 时间 | Date | ✅ | 日期 |
| 备注 | Text | ❌ | 附加说明 |
| 状态 | Select | ✅ | 活跃 / 已删除 |

## Integration 设置
1. 创建后复制 Token（格式：`secret_xxx`）
2. 在数据库中点击 `...` → `Add connections` → 添加 Integration
3. 从数据库 URL 提取 Database ID（格式：`notion.so/{workspace}/{DATABASE_ID}?v=...`）

## 字段说明

### 分类（Text）
分类使用 **Text** 类型而非 Select，原因：
- 应用内可自由定义分类名称，无需在 Notion 侧维护选项列表
- 避免同步时分类名称不匹配导致的冲突
- 灵活适配不同用户的记账习惯

**示例值**：餐饮、交通、购物、工资、奖金、医疗、通讯等。

> 💡 **提示**：如果需要精确筛选功能，建议在应用内保持分类命名规范一致。
