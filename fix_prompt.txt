### 2. "SimpleCallback is not a functional interface" (21 errors)
SimpleCallback has 2 abstract methods (onSuccess + onFailure), can't use lambda.
Convert ALL SimpleCallback lambdas to full anonymous class:

Pattern to replace:
```java
new SimpleCallback<Type>() {
    result -> handleResult(result)
}
```
Replace with:
```java
new SimpleCallback<Type>() {
    @Override
    public void onSuccess(Type result) { handleResult(result); }
    @Override
    public void onFailure(int code, String msg) { }
}
```

### 3. "reference to failure is ambiguous" (17 errors)
Two failure() methods from different interfaces conflict. Convert to explicit anonymous class with @Override on all methods (onSuccess + onFailure).

### 4. "constructor NonThrowError cannot be applied to given types" (9 errors)
First read NonThrowError.java to find its constructor signatures. Then fix all call sites to match.

### 5. "ValueFormatter is not a functional interface" (3 errors)
Same as SimpleCallback - convert lambdas to anonymous class.
